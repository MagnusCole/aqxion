# Project Structure — AQXION

## Directory Layout

```bash
src/
├── styles/
│   ├── base.css
│   ├── components.css
│   ├── globals.css
│   ├── tokens/
│       ├── colors.css
│       ├── spacing.css
│       └── typography.css
├── components/
│   ├── primitives/
│   └── composables/
│       ├── layout/
│       ├── navigation/
│       ├── data-display/
│       ├── forms/
│       ├── feedback/
│       └── marketing/
├── sections/
├── lib/
├── utils/
├── app/
```

## Principles

- Functional organization, not by abstraction level.
- Simple, clear, and scalable.
- Visual tokens as single source of truth.
- Parametrizable and reusable sections.
